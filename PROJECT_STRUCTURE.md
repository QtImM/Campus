## Data Flow

1. **Scraping**: Raw HTML data is collected in `data/scraped/`
2. **Parsing**: HTML is converted to structured CSV in `data/parsed/`
3. **Consolidation**: Multiple data sources are merged in `scripts/consolidation/`
4. **Processing**: Final data is stored in `data/processed/` as JSON
5. **Application**: Processed data is consumed by the app through services

## Development Workflow

1. Add new scraping scripts to `scripts/scraping/`
2. Add raw HTML output in `data/scraped/`
3. Add parsing scripts in `scripts/parsing/`
4. Keep parsed CSV in `data/parsed/`
5. Use consolidation scripts to merge data
6. Final processed data goes to `data/processed/`
7. Update services to consume new data
